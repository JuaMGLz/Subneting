Método Avanzado para la División de Redes

Aprende a dividir redes usando VLSM (Máscara de Subred de Longitud Variable)

Conceptos Claves Antes de Empezar

Dirección IP

Es como el número de casa de un dispositivo en la red. Es única y lo identifica.

Máscara de Subred

Es la regla que define qué parte de la IP es la red y qué parte es el host.

CIDR (/24)

Es la forma corta de escribir la máscara de subred. El número indica la cantidad de bits para la porción de red.

1

Paso 1: Imagina un Pastel (Tu Red Original)

Tu red es como un gran pastel que tienes que cortar en rebanadas de diferentes tamaños para tus invitados (los departamentos de tu empresa).

192.168.1.0/24
2

Paso 2: Ordena las Rebanadas (Requerimientos de Hosts)

Para ser eficiente, primero dale la rebanada más grande al invitado que más necesita. Después sigue con los demás en orden descendente.

RR.HH. (50 hosts)
Ventas (20 hosts)
Gerencia (10 hosts)
Impresoras (2 hosts)
3

Paso 3: Determina los Bits para Host

Usa la fórmula mágica 2^n - 2 ≥ hosts para encontrar el número de hosts que se ajusta a tus necesidades. El valor 'n' representa la cantidad de bits que necesitas para acomodar a todos tus hosts.

Recuerda: La máscara de subred original es /24 (24 bits para la red). Si necesitas 'n' bits para los hosts, tu nueva máscara CIDR será 32 - n.
4

Paso 4: Calcula las Máscaras de Subred

Aplica la fórmula para cada departamento y determina su máscara correspondiente.

Hosts requeridos Potencia de 2 (2^n) Hosts disponibles (2^n - 2) Nueva Máscara de Subred
50 (RR.HH.) 2^6 = 64 62 /26 (32 - 6)
20 (Ventas) 2^5 = 32 30 /27 (32 - 5)
10 (Gerencia) 2^4 = 16 14 /28 (32 - 4)
2 (Impresoras) 2^2 = 4 2 /30 (32 - 2)
5

Paso 5: Asigna las Subredes

Empieza con la red original y asigna consecutivamente. La dirección de la siguiente subred es la dirección de broadcast de la subred anterior más 1.

RR.HH. (50 hosts)
Ventas (20 hosts)
Gerencia (10 hosts)
Impresoras (2 hosts)
Direcciones de Broadcast:
• RR.HH.: 192.168.1.63
• Ventas: 192.168.1.95
• Gerencia: 192.168.1.111
• Impresoras: 192.168.1.115
Rango: 192.168.1.113 - 192.168.1.114

Resumen del Método Avanzado

  1. Paso 1: Ordena los requerimientos de hosts de mayor a menor.
  2. Paso 2: Para cada requerimiento, usa la fórmula 2^n - 2 ≥ hosts para encontrar la potencia de 2 que más se ajuste.
  3. Paso 3: Resta la potencia de 2 a 32 para obtener la nueva máscara de subred.
  4. Paso 4: Asigna la primera subred (la más grande) a la dirección de red original.
  5. Paso 5: Para cada subred siguiente, calcula el salto y úsalo para encontrar la siguiente dirección de red.
  6. Paso 6: Repite hasta que todas las subredes hayan sido asignadas.

¡Y listo! Con el método VLSM puedes optimizar el uso del espacio de direcciones IP y crear subredes eficientes para cada departamento.
